# XYZ Books

***Ruby version 3.0.0***

***Ruby on Rails version 7.0.4***

## **Local commands**

**Clone repo**
```
git clone https://github.com/felix1251/xyz_books.git
```
**Install dependecies**
```
rails db:setup
```
**or equivalent**
```
rails db:create
```
```
rails db:mgrate
```
```
rails db:seed
```
