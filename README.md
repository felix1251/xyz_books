# XYZ Books

***Ruby version 3.0.0***

***Ruby on Rails version 7.0.4***

## Clone repo
```ruby
git clone https://github.com/felix1251/xyz_books.git
```
```ruby
cd xyz_books
```
## Install Dependencies
```ruby
bundle install
```
## **Setup DB**
```ruby
rails db:setup
```
**Or equivalent**
```ruby
rails db:create
```
```ruby
rails db:mgrate
```
```ruby
rails db:seed
```
## **Run local server**
```ruby
./bin/dev
```
